# stego
данная прога прячет/показывает текст с фото(стеганография)
как скачать?
1. скачать прогу с данного репозитория
2. pip install pillow numpy cryptography --break-system-packages
как использовать?
зашифровать: python3 stego.py embed входная_картинка.png выходная_картинка.png --password "пароль" --secret "текст_который_прячем"
расшифровать: python3 stego.py extract картинка.png --password "пароль"

min FAQ
A: оно шифрует текст в фотке?

B: Да

A: какая защита?

B: AES-256-GCM

A: как именно?

B: вот так:
1. генератор случайных чисел создает соль (salt, 16 байт) и вектор инициализации (nonce, 12 байт)
2. функция PBKDF2-SHA256 делает 600 000 итераций хеширования пароля с солью, чтобы получить стойкий 256-битный ключ высокое число итераций защищает от брутфорса (перебора паролей)
3. текст шифруется алгоритмом AES-GCM. Он не только шифрует, но и добавляет 16-байтную метку аутентификации (Tag), которая гарантирует, что данные не были изменены
4. формируется финальный пакет данных (payload): MAGIC-маркер ("NCLD") + Соль + Nonce + Шифртекст с тегом
